# Old Scripts Archive

## Purpose

This directory contains archived versions of CGAS module scripts preserved for reference and version history.

## Contents

Initial versions of module scripts from earlier development stages, saved with `.txt` extension to distinguish them from active production code.

## Why Archive?

These files represent important development milestones and are retained for:
- Historical reference
- Comparison with current implementations
- Documentation of design evolution
- Troubleshooting legacy issues

## Usage

**Do not execute these files.** They are stored as `.txt` for archival purposes only. Refer to the active module scripts in `/cgas/` for current, production-ready code.

## Active Modules

For the latest working versions, see:
- `/cgas/` - Current production modules
- `/Documents/` - Module documentation and guides

---

**Note:** These archived scripts may contain outdated dependencies, methods, or approaches that have been improved in newer versions.
